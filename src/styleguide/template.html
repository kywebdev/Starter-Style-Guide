Buttons

A basic button uses the following base structure and styling: 

<a href="" class="btn">
    Base Button
</a>

.btn {
    font-family: $karla;
    font-size: 24px;
    text-decoration: none;
    color: $white;
    padding: 25px 30px;
    border-radius: 10px;
    display: inline-block;
    position: relative;
    top: 0px;
    text-align: center;
    transition: none;

    &:hover, &:active, &:focus {
        top: 2px;
    }
    &:before, &:after {
        display: none;
    }
}

To display properly, a button must also include a modifier class that indicates the color of the button, like so:

Blue Button 
<a href="" class="btn btn--blue">
    Blue Button
</a>

.btn--blue {
    background-color: $blue;
    box-shadow: 0px 7px darken($blue, 10%);

    &:hover, &:active, &:focus {
        box-shadow: 0 4px darken($blue, 10%);
    }
}
